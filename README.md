# If No One Reads This

**If No One Reads This** is a self-produced personal writing blog built with **HTML, CSS, and JavaScript**. It serves as a digital journal where I can share essays, letters, reflections, journal entries, poems, stories, and thoughts in a calm, minimalist environment designed for immersive reading.

This project was created to showcase my front-end development skills while building a realistic blogging platform with an elegant, distraction-free user experience. Rather than focusing on complex visuals, the website prioritizes meaningful writing, clean typography, responsive design, and intuitive interactions.

## Features

* ✍️ Create, edit, and manage personal writings
* 📅 Customize publication dates and post details
* 🏷️ Organize posts with categories and tags
* 🔍 Search and filter writings
* 🌙 Dark and Light mode
* 📖 Reading progress bar
* 📱 Fully responsive across desktop, tablet, and mobile
* ⚡ Smooth animations and page transitions
* 🎨 Minimal, modern, and reader-focused interface

## Built With

* HTML5
* CSS3
* JavaScript (Vanilla)

## Purpose

This project is part of my portfolio and represents both my passion for writing and my journey as a front-end developer. **If No One Reads This** is more than a blog—it's a quiet space where words are given room to breathe, whether they're read by many or simply written because they needed to exist.
